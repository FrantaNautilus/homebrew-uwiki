# FrantaNautilus/uwiki

Homebrew tap for [uwiki](https://github.com/FrantaNautilus/uwiki).

## Install

```bash
brew tap FrantaNautilus/uwiki
brew install uwiki
```

## Update
```bash
brew update
brew upgrade uwiki
```

## Uninstall / Untap
```bash
brew uninstall uwiki
brew untap FrantaNautilus/uwiki
```
